                                          Loan Default Prediction System
                                          
Project Overview

The Loan Default Prediction System is a machine learning project that analyzes loan applicant data to 
identify potential loan defaulters. The project uses Exploratory Data Analysis (EDA), Linear Regression, and
Data Visualization techniques to understand borrower behavior and support better loan approval decisions.

Problem Statement

Financial institutions face significant risks when issuing loans because some borrowers fail to repay them on time. 
This project aims to analyze borrower information and identify factors that influence loan default.

Source: Kaggle
Format: CSV

Project Workflow

1. Data Collection
Load Lending Club loan dataset
Understand dataset structure

2. Data Cleaning
Handle missing values
Convert categorical variables
Fix data types

3. Exploratory Data Analysis (EDA)
Loan Status Distribution
Income Distribution
Interest Rate Analysis
Correlation Analysis
Default Behavior Analysis

4. Risk Analysis

Borrowers are analyzed based on:

Income
Employment Length
Loan Purpose
Home Ownership
Interest Rate


5. Predictive Modeling

A Linear Regression model is built using:

Annual Income
Interest Rate
Loan Term
Target Variable:
Loan Amount

6. Model Evaluation
R² Score
Prediction Analysis
Performance Evaluation

8. Dashboard Visualization

Interest Rate Trends
Loan Status Comparison

The project helps identify important factors influencing loan repayment behavior and supports better loan approval strategies.
